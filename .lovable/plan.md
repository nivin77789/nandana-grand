

# Nandana Convention & Banquet — Premium Event Venue Website

## Overview
A luxurious, fully animated multi-page website for a premium wedding & event venue in Bangalore. The design uses a royal wedding aesthetic with gold, ivory, deep maroon, and champagne tones. All enquiries route through WhatsApp — no backend needed.

---

## Pages & Navigation

### Navbar
- Elegant sticky transparent navbar that fills on scroll
- Logo on the left, navigation links on the right
- Links: Home, About, Venues, Gallery, Packages, Book Now, Contact
- Mobile: animated hamburger menu with full-screen overlay
- Dark/Light mode toggle

### Page 1: Home
- **Hero Section** — Full-screen cinematic background image with overlay. Heading: *"Where Elegance Meets Celebration"*, subheading, and two CTA buttons (Book Now, View Gallery). Animated scroll indicator at the bottom.
- **About Section** — Elegant two-column layout (image + text) describing the venue. Highlights: fully air-conditioned, premium interiors, grand events. Scroll-reveal animations.
- **Venue Highlights** — 8 icon cards in a responsive grid (Seating Capacity, Parking, Bridal Room, Dining Hall, Stage & Decoration, Power Backup, Prime Location, Easy Access). Hover animations on each card.
- **Animated Counters** — Events Hosted, Happy Families, Years of Service — numbers count up on scroll.
- **Testimonials Carousel** — Animated card carousel with star ratings and client quotes.
- **Instagram Feed Preview** — Grid of Instagram-style images linking to the Instagram profile, with a Follow button.
- **CTA Banner** — Full-width parallax section with "Ready to Book Your Dream Event?" and Book Now button.

### Page 2: Gallery
- Masonry grid layout with category filters (Weddings, Receptions, Engagements, Decorations)
- Lightbox modal for image preview with smooth zoom
- Placeholder images initially — you'll upload real venue photos

### Page 3: Virtual Tour
- Placeholder section for future 360° embedded tour
- Parallax background with elegant messaging

### Page 4: Packages
- Three elegant pricing cards: Wedding Package, Reception Package, Custom Event Package
- Each card lists key inclusions
- CTA button: "Enquire Now" (links to booking form)

### Page 5: Book Now / Availability
- Calendar date picker to select event date
- Enquiry form: Name, Phone, Event Type, Event Date, Guest Count, Message
- On submit: opens WhatsApp with pre-filled message including all form details
- Elegant confirmation animation after submission

### Page 6: Contact
- Quick contact form (also WhatsApp-based)
- Call buttons for both numbers (+91 94812 50259, +91 98450 30799)
- WhatsApp button
- Embedded Google Map with full address
- Address: Ullal Main Rd, Jnananjyothinagar, Bengaluru, Karnataka 560056

### Footer (all pages)
- Logo, quick links, social icons (Instagram), contact info
- Copyright notice
- SEO structured data (LocalBusiness schema)

## Floating Elements (all pages)
- **WhatsApp button** — floating bottom-right, opens WhatsApp chat

---

## Design & UX Details
- **Color palette**: Gold, ivory, deep maroon, champagne, with glassmorphism accents
- **Typography**: Playfair Display for headings, Poppins for body text (via Google Fonts)
- **Animations**: Scroll-reveal effects, smooth page transitions, micro-interactions on buttons, parallax backgrounds, hover effects on cards
- **Responsive**: Mobile-first design, optimized touch targets, hamburger navigation
- **Performance**: Lazy-loaded images, optimized assets
- **Accessibility**: ARIA labels, proper heading hierarchy, keyboard navigation
- **SEO**: Meta tags, LocalBusiness schema markup, semantic HTML

---

## Future-Ready Architecture
- Clean, reusable component structure for easy scaling
- Multi-language support can be added later
- Blog section can be added as a new route
- Admin/backend booking system can be layered on via Lovable Cloud when needed
